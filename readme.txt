This is an AI agent made for a competition at wartburg. It plays the Hunt the Wumpus game designed as an a study a long time ago. The agent explores the maze to avoid the pits and monster, gets the gold, and finds its way out. The agent can not directly observe the world, but rather it only knows when it is near the a monster or pit or gold.

To run, use the following command:

python test_logic_agent.py

If this fails, make sure you have all of the following packages installed:

python-2.6 python-tk swi-prolog pylogic
